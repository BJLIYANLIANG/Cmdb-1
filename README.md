## 一、应用模块：

### 1.1 仪表盘模块

* 监控仪表盘
* 资产仪表盘
* 网站状态仪表盘

### 1.2 资产管理模块

* 设备管理
    * 设备机柜图
    * 设备IP占用表
    * 应用树

### 1.3 电子单据模块

* 请示件
* 项目管理
* 故障单

### 1.4 办公管理模块

* 网络管理
* AD域管理
* 巡检系统

### 1.5 预定管理模块

* 演播室预定
* 会议室预定
* 录音式预定

### 1.6 运维系统模块

* Saltstack管理
* Dns管理
* Svn管理
* 发布系统
* 日志系统
* 定时任务

### 1.7 电子工作流模块

* 电子工作流程系统
* 电子工作流程绩效系统

### 1.8 权限管理模块

* URL权限
* 用户组权限

### 1.9 其他

* 知道创宇管理
* 蓝汛DNS管理

## 三、平台环境

* **python 2.7.1**
* **django 1.6**
* **mysql 5.25a** 
* **redis 3.1**
* **metronic 4.7**

## 四、平台说明

* **RHEL/CentOS支持6.x系列，7.0以上暂未测试**
* **uWSGI和Nginx作为Web容器是可选的环境，可以不部署**
* **其他软件的版本号在实际运行环境中如果不同，为了兼容性保持主版本号相同即可**
